# Paginator

### Описание
Шаблонный класс, который разбивает содержимое контейнера на несколько страниц.

### Входные параметры класса
Конструктор класса принимает на вход два итератора -  на начало и на конец контейнера, а также число страниц, на которое необходимо разбить контейнер.

### Запуск теста
Обязательно наличие компилятора g++ с поддержкой стандарта С++17 и выше, а также утилиты make.
Для запуска теста класса, необходимо перейти в папка make-test и вызвать make.
Также запуск теста можно осуществить из папка bin
